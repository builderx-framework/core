# Commands

## bx Command

```shell
$ bx --help
Usage: bx [options] [command]

BuilderX CLI

Options:
  -V, --version                                output the version number
  -h, --help                                   display help for command

Commands:
  hello-world                                  Prints hello-world
```

## bx hello-world

```shell
$ bx hello-world
```
